Projects

Posture Guard AI

Apr 2026
  • Problem: Address "Tech Neck" from prolonged screen time with a low-cost, real-time posture monitoring tool using standard RGB cameras.
  • Data: Self-curated multi-scenario video corpus (3 users, varied lighting/occlusions) for robustness validation.
  • Approach: MediaPipe Face Mesh (468 landmarks) + OpenCV solvePnP for Euler angle estimation; multi-threaded architecture with thread-safe DoubleBuffer for 25–30 FPS; dynamic calibration to normalize camera-angle bias.
  • Outcome: ~89.2% Pitch accuracy & ~94.6% Roll accuracy (F1≈0.846); stable Tkinter GUI running on consumer hardware under non-ideal conditions.
  • Contribution: Led full-stack development: designed multi-threading pipeline, implemented PnP pose math & differential calibration logic, built real-time GUI, conducted robustness evaluation.

FinBERT Financial Sentiment Analysis

Mar 2026
  • Problem: Financial text classification suffers from domain nuance and class imbalance; need to systematically compare loss functions and hyperparameters for optimal generalization.
  • Data: Merged 3 Kaggle datasets (Twitter Financial News, NIFTY Headlines, FinancialPhraseBank); stratified 80/20 split with source-tagged preprocessing.
  • Approach: Fine-tuned ProsusAI/FinBERT via Hugging Face; executed full-factor grid search (32 configs: 2 loss × 4 LR × 4 BS); implemented Weighted CE & Enhanced Focal Loss with AdamW, warmup, gradient clipping, F1-based early stopping.
  • Outcome: Best config: Macro F1 89.92% / Acc 88.79%; paired t-test (p=0.789) proved Weighted CE ≈ Focal Loss for moderate imbalance; identified annotation ambiguity as primary error source.
  • Contribution: Co-developed hyperparameter grid search pipeline in PyTorch; implemented evaluation metrics (Macro F1, balanced accuracy) and paired t-test for statistical significance; authored convergence visualization and error analysis sections.
  • Link: GitHub Repo

ZARA Application Database Modeling
